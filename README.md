# Calm Markdown Writer (MVP)

Minimalist markdown writing app built with **React + TypeScript + Vite**.

## Features

- Local documents in `localStorage`
- Sidebar with create/rename/delete
- Write / Preview / Split modes
- Focus mode
- Light/Dark theme
- Import `.md` / `.txt`
- Export `.md` / `.txt`
- Autosave + save status (`Saved`, `Saving...`, `Unsaved changes`)
- Keyboard shortcuts:
  - `Ctrl/Cmd + S` save now
  - `Ctrl/Cmd + B` bold selection
  - `Ctrl/Cmd + I` italic selection
- Text stats: words, characters, reading time

## Run

```bash
npm install
npm run dev
```

## Build & typecheck

```bash
npm run typecheck
npm run build
```
